Необходимо сделать парсинг с ОЗОН по OZON SKU.
Страница для поиска выбирается по https://www.ozon.ru/product/{OZON SKU}/
Какие поля забираем
•	Название;
•	Категория (Автотовары/Запчасти для легковых авто/Двигатель/Система смазки/Масляные фильтры/MANN FILTER (находится вверху страницы);
•	Тип;
•	Страна;
•	Партномер или Артикул производителя (может называться и так, и так, ищем по вхождению пишем в одну ячейку);
•	Страна-изготовитель;
•	Изображения;
•	Описание;
•	Характеристики;

Программирование

Необходимо создать класс(ы) для парсинга страницы.
URL страницы передается в качестве параметра.
Должен возвращаться массив c результатами парса.
Изображения и характеристики должны возвращаться в виде вложенного массива.

Для старта заменить cookie в Provider на свои куки с сайта озона, вставить url в Provider запустить php.init для получения результата.
